# Stone Paper Scissor Game

A simple and interactive **Stone Paper Scissor** game developed using **HTML, CSS, and JavaScript**.  
The player competes against the computer, which makes random choices. The result is displayed instantly based on standard game rules.

---

## Features

- User-friendly interface
- Real-time result display (Win / Lose / Draw)
- Random computer choice generation
- Simple and clean design
- Beginner-friendly JavaScript logic

---

## Technologies Used

- **HTML** – Structure of the game
- **CSS** – Styling and layout
- **JavaScript** – Game logic and interactivity
